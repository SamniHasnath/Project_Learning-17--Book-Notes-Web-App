Book Notes Web App

A simple and modern Full Stack Book Notes application where users can add books, store reading notes, rate books, and manage their personal reading list.

Built using Node.js, Express, PostgreSQL, and EJS with a clean responsive UI.

🚀 Features
📖 Add books with title, author, ISBN, rating, and notes
✏️ Edit book details anytime
🗑️ Delete books
⭐ Rate books (1–10 system)
🔍 Sort by rating or newest
📱 Fully responsive UI
🌙 Dark mode support
🎨 Modern dashboard-style design
🛠️ Tech Stack
Frontend: HTML, CSS, EJS
Backend: Node.js, Express.js
Database: PostgreSQL
API: Open Library API (for book data)
Styling: Custom CSS (modern UI design)
Learning Purpose

This project helps understand:

CRUD operations
PostgreSQL queries
Express routing
API integration
EJS templating
UI/UX design principles
